# Inspiration Section

## Purpose
Provide trip ideas and templates to spark planning.

## Placement
- Dashboard below upcoming/past trips.

## Card content (draft)
- Title
- Subtitle/teaser
- Image
- Tags (season, style, pace)
- CTA (view template)

## Open questions
- Should inspiration link to a read-only trip template or a new-trip prefill?
- What metadata is most useful to filter ideas?
- Is this curated only, or can users save/share ideas?
